Reserved Operators That Map to Built-In Functions

In Tuppence, some operators are mapped to functions instead of being hardcoded as syntax.

Operator	Function Equivalent
==	eq?
!=	neq?
<	lt?
>	gt?
<=	lte?
>=	gte?
<=>	compare_to
+	add
-	sub
*	mul
/	div
%	mod
^	pow
[]	index
[]!	safe_index

Symbols That Are Not Reserved

cap	Retrieves array capacity
len	Retrieves array length

These functions are part of the core library, not the language syntax, 
meaning users can override or redefine them if needed.

Final Notes
	•	Identifiers cannot start with uppercase letters unless they are type identifiers.
	•	The backtick (`) is used for raw strings but not as an operator.
	•	Triple backticks (```) are reserved for multi-line string literals.
	•	The mut keyword is syntactic sugar for compiler-generated variable substitution.
